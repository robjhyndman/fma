# Dow-Jones index

Dow-Jones index, 28 Aug - 18 Dec 1972.

## Usage

``` r
dowjones
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 2.7.

## Examples

``` r
tsdisplay(dowjones)

```

# Employment figures in the motion picture industry

Monthly employment figures for the motion picture industry (SIC Code
78): Jan 1955 – Dec 1970.

## Usage

``` r
motion
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 7.9.

## References

"Employment and earnings, US 1909–1978", Department of Labor, 1979.

## Examples

``` r
plot(motion)

seasonplot(motion)

tsdisplay(motion)

```

# White noise time series

White noise time series with 36 values.

## Usage

``` r
wnoise
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chpater 7.

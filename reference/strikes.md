# Number of strikes

Number of strikes in the US from 1951 to 1980.

## Usage

``` r
strikes
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 7.4

## References

Brockwell and Davis (1991)

## Examples

``` r
tsdisplay(strikes)

```

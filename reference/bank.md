# Mutual savings bank deposits

Deposits in a mutual savings bank in a large metropolitan area.

## Usage

``` r
bank
```

## Format

Data frame containing the following columns:

- EOM:

  End of month balance

- AAA:

  Composite AAA bond rates

- threefour:

  US Government 3-4 year bonds

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapter 6.

## Examples

``` r
plot(bank)

```

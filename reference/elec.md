# Electricity production

Australian monthly electricity production: Jan 1956 – Aug 1995.

## Usage

``` r
elec
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapters 1–2, 7.

## Examples

``` r
plot(elec)

seasonplot(elec)

tsdisplay(elec)

```

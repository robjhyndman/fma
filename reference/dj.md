# Dow-Jones index

Dow-Jones index on 251 trading days ending 26 Aug 1994.

## Usage

``` r
dj
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapter 7.

## References

Brockwell and Davis (1996)

## Examples

``` r
tsdisplay(dj)

```

# Average temperature

Average monthly temperature in Paris.

## Usage

``` r
paris
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 2.1.

## Examples

``` r
plot(paris)

seasonplot(paris)

tsdisplay(paris)

```

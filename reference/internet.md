# Number of internet users

Number of users logged on to an internet server each minute over a
100-minute period.

## Usage

``` r
internet
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapter 7.

## Examples

``` r
tsdisplay(internet)

```

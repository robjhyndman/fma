# Canadian unemployment rate

Canadian unemployment rate as a percentage of the civilian labor force
between 1974 and the third quarter of 1975.

## Usage

``` r
canadian
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 4.1.

## Examples

``` r
plot(canadian)

```

# Condensed milk

Manufacturer's Stocks of evaporated and sweetened condensed milk.

## Usage

``` r
condmilk
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 7.5.

## Examples

``` r
plot(condmilk)

seasonplot(condmilk)

tsdisplay(condmilk)

```

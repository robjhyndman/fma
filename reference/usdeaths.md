# Accidental deaths in USA

Monthly accidental deaths in USA.

## Usage

``` r
usdeaths
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercises 2.3 and 2.4.

## Examples

``` r
plot(usdeaths)

seasonplot(usdeaths)

tsdisplay(usdeaths)

```

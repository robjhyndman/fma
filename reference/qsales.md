# Sales data

Quarterly exports of a French company in thousands of francs.

## Usage

``` r
qsales
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 3.7 and Table 4-7.

## Examples

``` r
plot(qsales)

```

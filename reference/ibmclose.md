# Closing IBM stock price

Daily closing IBM stock price.

## Usage

``` r
ibmclose
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 7.2.

## References

Box, Jenkins and Reinsell (1994) *Time series analysis: forecasting and
control*, 3rd edition, Holden-Day: San Francisco.

## Examples

``` r
tsdisplay(ibmclose)

```

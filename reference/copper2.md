# Copper prices

Yearly copper prices for 14 consecutive years (in constant 1997
dollars).

## Usage

``` r
copper2
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapter 9.

## Examples

``` r
plot(copper2)

```

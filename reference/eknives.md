# Sales of electric knives

Sales of electric knives: Jan 1991 - April 1992.

## Usage

``` r
eknives
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 4.2.

## Examples

``` r
plot(eknives)

```

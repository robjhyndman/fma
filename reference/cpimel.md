# Consumer price index

Quarterly CPI (consumer price index) for Victoria: Q1 1980 to Q2 1995.

## Usage

``` r
cpimel
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Exercise 8.7.

## Examples

``` r
tsdisplay(cpimel)

```

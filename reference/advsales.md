# Sales volume and advertising expenditure

Sales volume and advertising expenditure for a dietary weight control
product.

## Usage

``` r
advsales
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapter 8.

## References

Blattberg and Jeuland (1981).

## Examples

``` r
plot(advsales)

```

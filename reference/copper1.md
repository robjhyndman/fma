# Copper prices

Monthly copper prices for 28 consecutive months (in constant 1997
dollars).

## Usage

``` r
copper1
```

## Format

Time series data

## Source

Makridakis, Wheelwright and Hyndman (1998) *Forecasting: methods and
applications*, John Wiley & Sons: New York. Chapter 9.

## Examples

``` r
plot(copper1)

```
